JOB TITLE: Senior Software Engineer

COMPANY: Innovation Tech Labs

JOB TYPE: Full-time

Innovation Tech Labs is a fast-growing technology company building next-generation
cloud solutions for enterprise customers. We value innovation, collaboration, and

JOB DESCRIPTION:
We are seeking an experienced Senior Software Engineer to join our engineering team.

and contribute to architectural decisions.

- Collaborate with cross-functional teams (Product, Design, QA)
- Write clean, maintainable, and well-tested code
- Conduct code reviews and mentor junior developers
- Participate in system design and architectural discussions
- Optimize application performance and scalability
- Implement CI/CD pipelines and DevOps best practices

REQUIRED QUALIFICATIONS:
- 5+ years of software development experience
- Strong proficiency in Python or Node.js
- Experience with cloud platforms (AWS, Azure, or GCP)
- Solid understanding of microservices architecture
- Experience with SQL and NoSQL databases
- Proficiency with Git and version control
- Strong problem-solving and analytical skills
- Excellent communication and teamwork abilities

PREFERRED QUALIFICATIONS:
- Experience with Kubernetes and Docker
- Knowledge of React or Vue.js for frontend work
- Familiarity with event-driven architecture
- Experience with message queues (RabbitMQ, Kafka)
- Understanding of security best practices
- Open source contributions

TECHNICAL SKILLS:
- Languages: Python, JavaScript, TypeScript
- Frameworks: Django, Flask, Express.js
- Cloud: AWS (EC2, Lambda, S3, RDS)
